## Ce qui m'intéresse

Au-delà de l'administration quotidienne, ce projet est avant tout motivé par ma curiosité technique.

J'aime comprendre comment les différentes briques d'un système d'information fonctionnent, interagissent entre elles et contribuent à la sécurité globale de l'infrastructure.

À travers Active Directory, je cherche notamment à mieux comprendre :

* la gestion des identités ;
* la gestion des privilèges ;
* les mécanismes d'authentification ;
* les relations de confiance ;
* les modèles de délégation ;
* les erreurs de configuration les plus fréquentes ;
* les techniques utilisées par les attaquants pour compromettre un environnement Windows.

L'objectif n'est pas simplement d'administrer un annuaire, mais de comprendre en profondeur son fonctionnement afin d'identifier les risques, les points d'amélioration et les bonnes pratiques permettant de renforcer la sécurité du système d'information.

Cette démarche d'apprentissage continu me permet d'élargir ma vision de l'infrastructure et de développer une approche plus globale de l'architecture et de la cybersécurité.
